# css 文本超出省略号显示

## 单行

```
.ellipsis{
  width:500px;
  overflow:hidden;
  text-overflow: ellipsis;
  <!-- 文字超出不换行 -->
  white-space: nowrap;
 }
```

## 多行

```
.muti-ellipsis{
  display: -webkit-box;
  word-break: break-all;
  -webkit-box-orient: vertical;
  // 需要显示的行数
  -webkit-line-clamp: 3;
  overflow: hidden;
  text-overflow: ellipsis;
}
```
