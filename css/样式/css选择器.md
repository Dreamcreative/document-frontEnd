# css 选择器

  1. 通用选择器 *
  2. 元素选择器 elementname
    1. p
    2. span
    3. ......
  3. 类选择器 .classname
  4. ID选择器 #idname
  5. 属性选择器
    1. p[class='xxx']
    2. p[attr=value]
  6. 分组选择器 A, B
  7. 后代选择器 A B
  8. 直接子代选择器 A>B
  9. 一般兄弟选择器 A~B
  10. 相邻兄弟选择器 A+B
  11. 伪选择器
    1. 伪类选择器 p:active
    2. 伪元素选择器 p::first-line
