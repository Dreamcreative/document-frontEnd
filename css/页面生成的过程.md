## 浏览器渲染页面的过程

### 页面每次渲染都会按照这几个步骤进行。
Recaculate style 计算样式 -> Layer 重排->paint 重绘 ->Compose Layer  生成合成层->渲染
 而开启3D加速会跳过页面 Recaculate style 、 Layer、Paint 。直接在GPU中进行合成层的生成
