# 原型链图解

## 每个对象obj 都有一个私有属性（__proto__）
指向它的构造函数的原型对象（prototype）,
该原型对象也有一个（__proto__）,
层层向上直到一个对象的原型对象(Object.prototype) 为null 。 
__proto__连成的链成为原型链

## Function.prototype.bind 生成的函数没有 prototype属性，
因为生成的是一个 怪异的对象

## 每个函数都会默认的关联另一个对象 prototype，
