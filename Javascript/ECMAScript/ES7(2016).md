# ES7(2016)

1. Array.prototype.includes()

> 用来判断一个数组是否包含指定的值，包含 true，不包含 false

> arr.includes(a, fromIndex)

   1. a: 查找的元素值
   2. fromIndex: 从 fromIndex 索引处开始查找 a。默认 0 ，

2. 指数操作符 `**`

> 等效于 `Math.pow()`,不同的是 `**` 可以接收 BigInt 类型

```js
// 1024
2**10;
Math.pow(2,10)

// 256
2**2**3;
Math.pow(2,Math.pow(2,3));
2**(2**3);
```
